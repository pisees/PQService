public Task<HttpResponseMessage> EnqueueAsync([FromBody] IEnumerable<Item> items, [FromUri] short queue = null, [FromUri] int leaseSeconds = -1, [FromUri] int expirationMinutes = -1)
Parameters
Type Name Description
System.Collections.Generic.IEnumerable<Item> items

Array of MySampleItem instances to enqueue.
System.Int16 queue

Queue to add the item into.
System.Int32 leaseSeconds

Number of seconds for the default lease duration for each item in the set of items. Defaults to the lease duration configured for the queue.
System.Int32 expirationMinutes

Number of minutes for the expiration for each item in the set of items. Defaults to the expiration duration configured for the queue.
Returns
Type Description
System.Threading.Tasks.Task<System.Net.Http.HttpResponseMessage>

HttpStatusCode.NoContent (200) if all items were successfully enqueued with a QueueItem<string> response item for each item in the body. HttpStatusCode.BadRequest (400) if no items were passed in the body. HttpStatusCode.BadRequest (417) if the queue, lease or expiration are not within their valid ranges. HttpStatisCode.InternalServerError (500) an unknown error has occurred.
Remarks

This is not an idempotent operation.

Examples
